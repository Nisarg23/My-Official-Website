This is my gui implemnetaion of the sudoku puzzle game using the pygame module in Python. The randomizer uses mulitple sorting algorithms to generate a random board. The solver uses a backtracking recursive algorithm which searches for the closet empty cell and adds a valid number. If there are no valid numbers availbe the alogrithm backtracks to a previous cell and modifies it and repeats the process until the board is solved.
